2、前端

2.1、前端集成

1、先获取所有字典类型

    接口地址：http://govtest.snkoudai.com/api/foundation/dict/listOfDictType


入参：

{
    "keyword": "",
    "pageNo": 1,
    "pageSize": 10
}

序号 字段 类型
 说明
1 keyword
 String 字典名称搜索，为null或者空查询全部
2 pageNo
 int 页码
3 pageSize
 int 条数


响应：

{
    "respCode": 0,
    "respDesc": "成功",
    "curDate": 1656643536811,
    "obj": {
        "list": [
            {
                "id": 1,
                "dictName": "所有权性质",
                "dictType": "1",
                "createTime": null,
                "updateTime": null
            },
            {
                "id": 2,
                "dictName": "地块类别",
                "dictType": "2",
                "createTime": null,
                "updateTime": null
            }
        ],
        "totalCount": 22
    },
    "success": true
}
序号
 字段
 类型
 说明
1
 id Long id主键
2
 dictName String 字典名称
3 dictType String 字典类型




示例图片

在中台，我们已经集成了字典表数据



2、根据字典类型去查询字典数据

接口地址：http://govtest.snkoudai.com/api/foundation/dict/listByDictType


入参：

{
    "dictType": "8"
}
序号 字段 类型
 说明
1
 dictType String 字典类型



响应：

{
    "respCode": 0,
    "respDesc": "成功",
    "curDate": 1656644174450,
    "obj": [
        {
            "id": 73,
            "dictType": "8",
            "dictLabel": "男",
            "dictValue": "1",
            "sortNum": 0,
            "createTime": null,
            "updateTime": null
        },
        {
            "id": 74,
            "dictType": "8",
            "dictLabel": "女",
            "dictValue": "2",
            "sortNum": 0,
            "createTime": null,
            "updateTime": null
        }
    ],
    "success": true
}


序号 字段
 类型
 说明
1
 id Long id主键
2 dictType String 字典类型
3 dictLable String
 字典标签
4 dictValue String
 字典键值
5 sortNum int 排序字段
